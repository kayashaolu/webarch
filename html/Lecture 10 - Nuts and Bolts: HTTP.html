HTTP Response Codes

  • Numbers are unambiguous, can be read by machines
  • Always return a response code
  • Return text as well for human readability

HTTP Response Numbers

  • 200's: OK
  • 300's: Redirect
  • 400's: Client Errors
  • 500's: Server Errors
